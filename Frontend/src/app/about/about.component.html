FAQ de MyGainz

Ici vous retrouverez les réponses aux questions les plus communes

Tout d'abord, il faut savoir que MyGainz est un projet réalisé dans le cadre d'un projet universitaire dans la matière développement multimédia.

Qu'est-ce que MyGainz

MyGainz est une plateforme visant originellement à tracker ses calories, mais n'a pas eu le temps d'être complétée, elle permet donc uniquement de gérer une liste de plats

Quelle est la fonctionnalité Android ?

Cette application implémente un tracker de course permettant de donner plusieurs données sur une course.
